Parameters and filters

The region resource does not support any URL parameter or filter besides format selection and language negotiation.

Region item data structure

Since the self link is relying on the geonames API, region do not have individual resource. The list is designed to allow client to lookup all the regions available and, for example, filter accordingly.

The region resource is a list of items. As a consequence, the overall data structure will follow the list structure as defined in the formatting guidelines.
