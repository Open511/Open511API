--- layout: page title: Jurisdiction presentation : a : name : purpose value : The jurisdiction represents any government entity (city, state, agency) that wants to publish open511 data. It is a functional concept used to provide some metadata about events like contact information b : name : method value : GET representation : a : name: self mandatory: Mandatory type: Link desc:

Self link to the current resource using atom link structure with self attribute.

<atom:link rel="self" href="http://ville.montreal.qc.ca/open511/jurisdiction/ville-montreal/" />
b : name: parent type: Link mandatory: Mandatory desc:

Self link to the parent resource using atom link structure with parent attribute

c : name: name type: free text mandatory: Mandatory desc:

Name of the jurisdiction.

d : name: email type: Free text mandatory: Mandatory desc:

Valid email that can be used to contact the department or the person in charge of the data provided by the Open511 API

e : name: url type: Link mandatory: Optional desc:

Human readable resource providing detail about the Open511 service and the jurisdiction covered.

f : name: phone type: Free text mandatory: Optional desc:

Phone number to contact the department or the person in charge of the data provided by the Open511 API

g : name: description type: Free text mandatory: Optional desc:

Free text that can be used by a client application to provide some information about the Open511 service and the jurisdiction covered.

h : name: area type: GML notation mandatory: Mandatory desc:

Geographical coverage of the jurisdiction. This field only accepts a polygon or a multipolygon. Issue 14

Parameters and filters

The discovery resource does not support any URL parameter or filter besides the format selection and the language negociation.

Example

Example of the XML and JSON representations of the discovery resource

  • XML
  • JSON
<open511
    xmlns:atom="http://www.w3.org/2005/Atom"
    xmlns:gml="http://www.opengis.net/gml"
    xml:lang="en"
    xml:base="http://api.open511.info/"
>  
  <jurisdiction>
    <atom:link rel="self" href="http://my.city.gov/open511/jurisdiction/my-city/" />
    <atom:link rel="up" href="http://my.city.gov/open511/" />    
    <name>My City</name>
    <email>mayor@my.city.gov</email>
    <phone>+1 555-555-0000</phone>
    <description>Official traffic data (construction) from My City</phone>
    <atom:link rel="description" href="http://my.city.gov/open511/presentation.html" />
    <area>
      <gml:Polygon srsName="EPSG:4326">
        <gml:coordinates>-71.17,47.33 -71.15,47.36 -71.10,47.35 -71.20,47.40</gml:coordinates>
      </gml:Polygon>
    </area>
    <license>
      <atom:link rel="copyright" href="http://my.city.gov/opendatalicense/"/>
      <licenseName>CC-BY</licenseName>
      <conditions>
        <condition>ATTRIBUTION</condition>
        <condition>SHAREALIKE</condition>
      </conditions>
    </license>
    <languages>
      <language>fr</language>
      <language>en</language>
    </languages>
  </jurisdiction>
</open511>
